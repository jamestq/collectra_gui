API Reference
=============

Core API
--------

The API module provides the pywebview interface for the annotation display application.

Key Components
~~~~~~~~~~~~~~

- **Window Management** - Control the application window and lifecycle
- **Event Handling** - Handle user interactions and data updates
- **Data Interface** - Bridge between Python backend and JavaScript frontend
