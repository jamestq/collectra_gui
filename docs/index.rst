Collectra GUI
=============

.. image:: https://img.shields.io/badge/python-3.12+-blue.svg
   :alt: Python Version

.. image:: https://img.shields.io/endpoint?url=https://gist.github.com/jamestq/3d9df571edfc07dcd67e9fa6cdac19a7/raw/coverage-badge.json

A modern annotation display and editing tool built with pywebview. Visualize annotation lineage and interactively edit annotation data with ease.

Features
--------

- **Interactive Editing** - Edit annotations in real-time with a user-friendly interface

- **Lineage Visualization** - Track and visualize annotation relationships and history

- **Cross-Platform** - Works on Windows, macOS, and Linux via pywebview

Quick Start
-----------

Install via pip and then run:

.. code-block:: bash

   pip install collectra_gui
   collectra_gui
