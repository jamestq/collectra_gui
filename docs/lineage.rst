Lineage Display
===============

Overview
--------

The lineage display module provides graph structures and visualization logic for tracking annotation relationships over time.

Features
~~~~~~~~

- **Graph Visualization** - Display annotation relationships as interactive graphs
- **Lineage Tracking** - Follow the history and evolution of annotations
- **Relationship Mapping** - Understand parent-child and sibling relationships
